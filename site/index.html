About this Project

A fully automated, secure static site on AWS — S3 (private) + CloudFront (OAC) with WAFv2, strict security headers (CSP/HSTS/COOP/COEP/Permissions-Policy), and CI/CD via GitHub Actions (OIDC). Infrastructure is defined in Terraform, including budgets, lifecycle policies, and teardown-safe settings.

  • Security: OAC, HTTPS-only, managed WAF rules, strict headers
  • Ops: Terraform state locking, CI scans (fmt/validate, TFLint, Checkov)
  • Perf/Cost: cache policies for HTML vs assets, compression, PriceClass 100
